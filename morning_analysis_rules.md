# 朝一分析ルール定義

## 朝一の定義

**朝一（設定変更後）** とは、その日の1回目（0スルー）の当たりから天国チェーンに突入するまでの期間を指す。

### 詳細条件

| 項目 | 定義 |
|------|------|
| 対象データ | その日・その台の最初のチェーン（朝イチ連=TRUE） |
| 開始 | 1回目の初当たり（連チャン位置=1、朝イチ連=TRUE） |
| 終了 | 天国チェーン（連チャン回数>=2）に突入、または朝一スルー終了 |

### 分析項目

1. **朝一初当たりゲーム数分布**
   - 1～50G, 51～100G, 101～150G, ... の50G刻み
   - 件数、振分（%）、累積当選率を表示

2. **朝一天国移行率**
   - 朝一チェーンが天国（2連以上）になる確率

3. **朝一スルー回数分布**
   - 朝一から天国移行までのスルー回数分布

## データ取得条件

```
ChainData!E列 = TRUE（朝イチ連）
Data!K列 = TRUE（朝イチセッション）
Data!J列 = 1（連チャン位置が1 = 初当たり）
```
